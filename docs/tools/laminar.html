Frequently asked questions

Is Laminar open source?

Yes. Licensed under Apache-2.0.

Can I self-host Laminar?

Yes — Laminar can run in your own infrastructure.

Is Laminar OpenTelemetry-native?

Yes — Laminar speaks OpenTelemetry (OTLP) natively, so there is no proprietary-SDK lock-in.

How much does Laminar cost?

Freemium. OSS self-host (Docker) free; cloud free tier + paid plans
